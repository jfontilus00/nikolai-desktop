🚀 QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════════

1. Build and Run:
   pnpm dlx @tauri-apps/cli@1.5.9 dev

2. Install MCP Server (example):
   npm install -g @modelcontextprotocol/server-tools

3. Configure in UI:
   - Open Tools tab
   - Command: node
   - Args: C:\path\to\mcp-server.js --stdio
   - Click Connect
   - Click List tools

4. Use in Chat:
   /tool calculator {"expression": "2 + 2"}
