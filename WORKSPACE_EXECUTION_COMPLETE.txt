================================================================================
BATCH OPERATIONS:
================================================================================

Apply:
  - Creates batch directory with timestamp as ID
  - Backs up existing files before overwriting
  - Writes new files or creates new ones
  - Records metadata in batch.json
  - Logs all operations to manifest.jsonl

Rollback:
  - Restores backed-up files for existing files
  - Deletes newly created files
  - Can rollback by specific batch ID or latest
  - Updates manifest with rollback operations
